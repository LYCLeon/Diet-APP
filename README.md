### 主題 ###

在這個APP裡面，你可以選擇在一餐之中，吃了甚麼食物並攝取多少卡洛里；在一天之中，你走了幾步，消耗多少卡洛里，或者你選擇了甚麼運動去消耗卡洛里，設立目標、持之以恆並達成減重的效果。
## USER STORY ##

###使用情節1###

蒐集食物卡洛里資料
建構天數:7  剩餘天數:35
###使用情節2####
用戶登入或註冊
使用Facebook ,Twitter ...

###使用情節3###

###使用情節4

###使用情節5

###使用情節6
